# APIX 2분 발표 스크립트 (한국어)

## 2분 발표 스크립트

**[0:00–0:15]**
안녕하세요, 심사위원 여러분. 저희는 **[TEAM NAME]**입니다.
저희 팀이 만든 프로젝트는 **APIX**입니다.

**APIX**는 웹 API에서 스테이블코인 결제를 바로 처리할 수 있게 해주는,
**Avalanche L1 기반의 x402 결제 SDK**입니다.
아이디어는 간단합니다.
“결제는 API 호출만으로 끝내고, 결과는 온체인에서 검증 가능하게 만들자.”

**[0:15–0:40]**
현재 결제는 여전히 경험을 깨뜨립니다.
게임에서는 결제창 리디렉션과 플랫폼 의존 흐름이 몰입을 끊고,
AI 에이전트는 신용카드·구독 모델을 쓰지 못해 세분화된 API 결제를 못 합니다.
결국 팀들은 “어떤 결제가 어떤 액션과 정확히 매칭되는지”를
오프체인 로그로 일일이 추적해야 합니다.

**[0:40–1:10]**
**APIX**가 이걸 바꿉니다.
개발자는 서버에 미들웨어를 붙이기만 하면,
요청 경로가 x402을 지원하도록 변환됩니다.
예를 들어 `/buy/sword`를 호출하면 서버가 **402 Payment Required**를 반환하고,
클라이언트는 서명된 결제 정보를 담아 재요청합니다.
그 순간 APIX는 결제 영수증을 Avalanche L1에 기록하고,
`payer`, `item`, `amount`, `nonce`가 함께 담긴 신뢰 가능한 증거를 남깁니다.

**[1:10–1:40]**
우리는 이를 받아 즉시 검증해, 게임 아이템 지급이나 API 응답을 즉시 실행합니다.
응답이 실패하면 과금이 일어나지 않습니다.
온체인 전송과 실제 제공값의 불일치도 사라집니다.
그리고 별도 정산 시스템을 직접 만들 필요도 없어집니다.

**[1:40–2:00]**
대상은 게임 스튜디오, 디지털 자산 마켓플레이스, 그리고 AI 에이전트 빌더입니다.
게임에서는 결제와 인게임 자산 전달이 자동 동기화되고,
AI에서는 호출 단위의 경제가 현실화됩니다.
우리는 APIX로 **결제를 ‘재화 이동’이 아니라 ‘서비스 호출의 정식 동작 단위’**로 바꾸려 합니다.
감사합니다.
